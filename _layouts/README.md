# General Quiz Demo #2

This test version is not connected to TacoCat branding or content.

## Features
- 10 general-knowledge questions
- 60-second timer
- Name or username field
- Hidden participant score
- Results sent to Supabase
- Saves score, answers, completion time, timeout status and submission date

## Setup
1. Open Supabase SQL Editor.
2. Run `supabase-setup.sql`.
3. Open Project Settings → API.
4. Add the Project URL and public anon key to `supabase-config.js`.
5. Upload all files to the test website.
6. Open `index.html`.

Never place the Supabase service-role key in browser code.
