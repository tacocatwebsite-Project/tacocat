IMPORTANT:
This practice build stores results in the browser's localStorage. It is suitable for testing the quiz and panel on the SAME iPhone/browser.
It does not collect results from different participants' phones into one shared dashboard.

For the official event, connect the quiz to Supabase, Firebase, or another database so every participant submission appears in your private panel.
